## ✨ 핵심 기능

### 1. AI 생활비 코치 (Budget Coach)
단순 놀거리 추천이 아니라 **월 예산과 소비 패턴까지 관리**합니다.
남은 생활비·이번 주 지출을 분석해 *"이번 주말은 최대 3만원 사용을 추천해요"* 처럼 적정 예산을 먼저 제안합니다.
과소비한 주에는 **무료 전시·공원·산책·팝업** 등 저비용 코스를, 예산이 여유로우면 더 다양한 경험을 우선 추천합니다.

### 2. 오늘의 AI 브리핑 (Proactive Briefing)
검색하지 않아도 AI가 먼저 제안합니다. 날씨·예산·저장한 팝업·시간대를 종합해
*"오늘은 비가 와서 실내 코스를 추천드려요"*, *"저장한 팝업이 오늘 마지막 운영일이에요"* 와 같이 안내합니다.

### 3. 10초 코스 설계 → 만족도 타임라인
예산 · 동행 · 위치 · 취향 · 실내/야외만 고르면, AI가 예산·날씨·시간·혼잡도·실시간 할인·팝업·무료 전시를
종합해 **만족도 최적의 하루 코스**를 타임라인으로 설계합니다. (예상 지출 / 만족도 점수 / 예산 사용률 포함)

### 4. 실데이터 소셜 랭킹 (Social Proof)
*'1만원으로 놀기'*, *'3만원 성수 데이트'* 처럼 또래가 만든 예산 코스를 공유·저장·따라가기.
**세그먼트별 실데이터 랭킹**(예: 20대 여성·혼자·2만원 이하 만족도 TOP, 20대 커플·5만원 이하 평점 4.9 TOP)으로 신뢰도를 높입니다.

### 5. AI 개인화 학습 (Personalization)
예산뿐 아니라 **행동을 학습**합니다 — 실내 선호, 카페 자주, 혼자 놀기, 팝업 저장, 오후 활동 등.
쓸수록 추천 정확도가 올라가는 것을 *정확도 게이지*로 시각화합니다.

### 6. 게이미피케이션 (Gamification)
*1만원 챌린지 · 3만원 데이트 · 이번 달 절약왕 · 가성비 탐험가* 등 미션·배지로
즐기며 절약하고, 자연스럽게 UGC와 커뮤니티를 활성화합니다.

## 🧭 화면 구성 (IA)

```
머니코스 (PWA)
├─ /            온보딩 · 서비스 소개 (한 문장 정의 · 문제 · 차별점 · CTA)
├─ /home        오늘 대시보드 — AI 브리핑 · 생활비 코치 · 추천 코스 · 학습된 취향 · 챌린지
├─ /plan        코스 설계 — 예산·동행·위치·취향·시간 입력 → AI 생성
├─ /result      AI 설계 코스 — 만족도/지출/시간 · 타임라인 · 저장/공유/다녀왔어요/다시추천
├─ /explore     탐색 — 추천 피드(예산 필터) · 실데이터 랭킹(세그먼트별 TOP)
├─ /course/[id] 코스 상세 — 코스 순서 · 후기 · 비슷한 코스 · 따라가기
├─ /saved       저장함 — 내가 만든 코스 · 저장한 코스
└─ /profile     MY — 생활비 코치 설정 · 학습된 취향 · 챌린지 배지
```

하단 탭바(홈·탐색·저장·MY) + 중앙 FAB(코스 설계)로 **네이티브 앱과 같은 내비게이션**을 제공합니다.

## 🚀 실행 방법

### 사전 요구사항
- Node.js 18.18+ (권장 20+)

### 로컬 개발
```bash
npm install
npm run dev
# http://localhost:3000
```

### 프로덕션 빌드 & 실행
```bash
npm run build
npm start
```
> 서비스 워커(PWA 오프라인 캐시)는 **프로덕션 빌드에서만** 활성화됩니다.

### PWA 설치 (앱처럼 사용)
모바일 브라우저에서 접속 후 **‘홈 화면에 추가’** → 전체화면 standalone 앱으로 동작합니다.

### 아이콘 재생성 (선택)
```bash
npm run icons   # public/icons/* (sharp로 SVG→PNG)
```

## 🛠 기술 스택 & 설계 노트

- **Next.js 14 (App Router)** — 정적 프리렌더 + 클라이언트 인터랙션 혼합, Vercel 1-click 배포.
- **TypeScript** — 도메인 타입 1급 관리(`lib/types.ts`).
- **Tailwind CSS** — Apple/Linear/Vercel/Toss 톤의 미니멀 디자인 시스템(커스텀 토큰·애니메이션).
- **PWA** — `manifest.webmanifest` + 직접 작성한 서비스 워커(`sw.js`, app-shell precache · 네비게이션 network-first · 정적 SWR), 오프라인 폴백.
- **상태/영속화** — 외부 의존 없이 React Context + `localStorage`. 예산·취향·저장·배지가 새로고침 후에도 유지.
- **AI 엔진(데모)** — 실제 추천 로직을 모사한 결정론적 규칙 기반 엔진. 동행 적합도·분위기·실시간 시그널(할인/팝업/무료/혼잡도)·날씨 맥락·**개인화 학습**·**생활비 코치 절약 강도**를 종합 점수화하고, 예산 제약 안에서 카테고리 다양성을 고려해 코스를 조립합니다. 프로덕션에서는 장소/할인/팝업/혼잡도 API와 랭킹 모델로 대체됩니다.

> 데이터(장소·할인·랭킹)와 일부 지표는 **데모용 큐레이션/가설**이며, 기획서의 ‘부록 — 핵심 가정 & 검증 계획’에 검증 방법을 정리했습니다.

- **North Star**: 주간 완성된 코스 경험 수(Weekly Completed Course Experiences)
